# poistenieGM_bot

Бот для получения информации о медицинском страховании в Словакии.

## Запуск

1. Установите зависимости:
   ```
   npm install
   ```
2. Создайте файл `.env` и добавьте токен:
   ```
   BOT_TOKEN=ваш_токен_бота
   ```
3. Запустите бота:
   ```
   npm start
   ```

## Структура проекта

- `index.js` — точка входа, инициализация бота
- `handlers/basicHandlers.js` — обработчики команд и сообщений
- `.env` — переменные окружения (токен)
- `.gitignore` — исключения для git

## Расширение
Добавляйте новые обработчики в папку `handlers/` для масштабирования функционала.
